# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [2.0.0] - 2026-05-06

### Added

- Improved family onboarding flow for creating or joining families.
- Enhanced grocery item handling with smoother add/edit interactions.
- Better activity visibility for family-level grocery updates.

### Changed

- Faster startup and smoother splash-to-app transition during auth/session restore.
- Cleaner navigation behavior across family/member grocery views.
- More consistent grocery item status and priority presentation.

### Fixed

- Release bundling failure caused by invalid Hermes compiler path configuration.
- Stability issues in authentication/session restore and grocery item update flows.

## [1.0.0] - 2026-05-04

### Added

- Open-source Family Grocery List app with Expo + React Native.
- Firebase auth and Firestore-backed family grocery collaboration flows.
